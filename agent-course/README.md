## 학습 목표

- AI 에이전트의 기본 개념 이해
- LangChain을 사용한 에이전트 개발
- 다양한 에이전트 타입과 도구 활용
- 에이전트 개발의 모범 사례 학습
- 다양한 에이전트 프레임워크의 이해와 활용
- 실제 프로덕션 환경에서의 에이전트 구현
